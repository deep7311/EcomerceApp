User Routes

POST - /api/users/register (Use in Postman)

POST - /api/users/login (Use in Postman)

GET - /api/users/all (admin only)

GET - /api/users/:id (admin only)

DELETE - /api/users/:id (admin only, use in Postman)

PUT - /api/users/:id (admin only, use in Postman)

GET - /api/users/:id/profile (user only)

PUT - /api/users/:id/profile (user only, use in Postman)

PUT - /api/users/:id/password (user only, use in Postman)
